## Requisitos

### node
### npm
### IDE de preferencia

## Instalar dependencias

### `cd server`
### `npm install`
### `cd webapp`
### `npm install`

## Correr el servidor mock

### `cd server`
### `node index.js` 

## Correr el app de react

### `cd webapp`
### `npm start`
